- **Evidência**: `supabase/migrations/20250218000006_RESTO.sql:89-106`

- **Impacto**: Double-booking check silenciosamente falha. Clientes podem reagendar para slots ocupados
- **Correção**: Trocar `booking_time` por `start_time`
